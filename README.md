# Hackmol-5.0 Jalandhar

## Prerequisites
1. Node JS >= 18
1. Python >= 3.7
1. NPM >= 9

## Instalation Steps

1. Clone the repository
    ```sh
    git clone https://github.com/Ishanchauhan06/Hackmol-5.0.git
    ```

1. Navigate to the repository
    ```sh
    cd Hackmol-5.0
    ```
